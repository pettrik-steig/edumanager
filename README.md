# EduManager 2026

EduManager is a planned Java-based school management application for students,
learning groups, timetable context, room situations, and seating arrangements.

The project starts from lessons learned in the predecessor prototype
`Sitzplan`: use cases, object ownership, identity, copy semantics, persistence,
and application services should be made explicit from the beginning.

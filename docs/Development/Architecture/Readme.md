# Architecture Development

This directory contains architecture development material for EduManager.

The current focus is scenario-based information flow analysis. Architecture
work should start from concrete application situations and derive concepts,
requirements, and architecture questions from those situations.

Files:

- [goal.md](goal.md): current goal for the architecture process
- [domain-terms.md](domain-terms.md): central German domain terms and
  non-technical English descriptions
- [Scenarios](Scenarios/Readme.md): concrete application scenarios to analyze

# Scenario Analyses

This directory contains concrete application scenarios analyzed for
EduManager's architecture development.

Each scenario should stay close to a real practical work situation before it is
translated into requirements or architecture.

Current scenarios:

- [zuweisung-zu-klasse-lerngruppe-kurs.md](zuweisung-zu-klasse-lerngruppe-kurs.md):
  a student is assigned to a class, learning group, or course context
- [erster-kontakt-im-unterricht.md](erster-kontakt-im-unterricht.md): first
  classroom contact with a student from the teacher's perspective
